# Circular Knitting Production Calculator

This web app calculates:

- Needles
- Feeders
- Production Counter

## Formula

Needles = 3.14 × Gauge × Diameter  
Feeders = Diameter × 3 + 6  
Counter = (Kg × 16936517 × Count) ÷ Stitch ÷ Needles ÷ Feeders

## Run

Just open index.html in browser.
